# tatakabu
叩いてかぶってじゃんけんぽんアプリを作成<br>

## 概要
難易度は初級、中級、上級の三段階。違いは、相手の反応速度だけ、、、
じゃんけんに勝ったら、叩く、負けたらかぶってガードし、先に相手を3回たたいたら勝ち。
